**What these numbers mean** (same disclosure as `migration_audit.md`'s own per-worksheet table): Calculation Match % and Filter Match % are *measured* from this workbook's actual DAX translation and filter conversion results. Visual / Layout / Format Match % are *heuristics* from known mapping quality and detected signals, not a pixel-level comparison against the original Tableau render. Overall Match % is a weighted blend (30% visual, 30% calc, 20% filter, 10% layout, 10% format).
